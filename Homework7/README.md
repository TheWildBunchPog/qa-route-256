### Локальная база данных для стены сообщений

Образ
```

```

Запуск
```
docker pull gitlab-registry.ozon.dev/thewildbunch/qa-route-256:Homework
docker run -d -p 5000:5000 gitlab-registry.ozon.dev/thewildbunch/qa-route-256:Homework
```
